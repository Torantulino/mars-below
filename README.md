## 🎮 What Is This?

**Mars Below** is a faithful web recreation of XGen Studios' **Motherload** (2004) — the legendary Flash mining game where you drill beneath the Martian surface, harvest minerals, upgrade your pod, and discover the horrors that lurk in the deep.

**How to play:** Arrow keys or WASD. Drill down. Collect shiny things. Sell them. Buy upgrades. Go deeper. Don't run out of fuel.

## ⚡ What's Inside

- 🗺️ **Procedurally generated** Martian underground — 32×600 tile grid
- 💎 **10 mineral types** from Ironium ($30) to Amazonite ($500,000)
- 🔧 **6 upgrade categories** × 6-7 tiers each — drill, hull, engine, fuel, radiator, cargo
- 🔥 **Environmental hazards** — rocks, lava pockets, invisible gas pockets, marsquakes
- 📻 **12 narrative transmissions** — the story of Mr. Natas unfolds as you dig
- 👹 **Two-phase boss fight** against Satan in a business suit
- 🎵 **Procedural audio** via Web Audio API — zero audio file downloads
- 📱 **Mobile touch controls** with virtual D-pad
- 🌀 **Vibe Jam Portal** integration (webring)
- 💾 **Auto-save** to localStorage
- ⚡ **Instant load** — playable in under 500ms, zero external assets

**Single file. Zero dependencies. ~85KB total.**
